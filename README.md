# 🛒 Superstore Sales Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on the Superstore Sales dataset using Python. The objective is to clean the data, identify business trends, analyze customer purchasing behavior, and generate actionable insights through statistical analysis and visualizations.

---

## 🎯 Project Objectives

- Analyze overall sales performance.
- Identify top-selling and most profitable products.
- Compare sales across different product categories.
- Analyze customer purchasing patterns.
- Evaluate sales by region and city.
- Study yearly sales trends.
- Discover relationships between numerical variables using correlation analysis.

---

## 💡 Key Business Insights

- Technology was the highest revenue-generating product category.
- Canon imageCLASS 2200 Advanced Copier generated the highest sales and profit.
- The West region achieved the highest overall sales.
- New York recorded the highest city-wise sales.
- Sales showed continuous growth over multiple years.
- Strong relationships between numerical features were identified using correlation analysis.
